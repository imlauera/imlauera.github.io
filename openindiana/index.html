NO LO RECOMIENDO, ARCHLINUX ES LA MEJOR DISTRO.

Descargué el iso de openindiana para USB el text install y lo instalé en mi netbook sin querer descargué la versión de 32 bits.
Instalá usando todo el HDD

Al sistema lo tuve que bootear con vesa con los drivers que tenía me daba pantalla NEGRA.

su
pkg update (pesaba 769 MB)
pkg refresh --full
pkg publisher
pkg install tmux git dmenu firefox mate-desktop mate-session-manager xinit mate-panel caja marco mate-terminal xorg xorg-video-intel twm xterm xorg-video-vesa

Tmux me tomó 4 minutos, git me tomó 4 minutos también.
La instalación de firefox time pkg install firefox me tomó aproximadamente 6 minutos con 48 segundos.

Los nombres de HDD en OpenIndiana son /dev/rdsk/c2t0d0 no son /dev/sda.

Para ver las particiones, usa el comando format.

prtvtoc /dev/rdsk/c0t0d0s2
zfs list
zpool status

Instalé la versión de 32 bits por error y no tenía ni mpv ni ffmpeg XD.

svcadm enable algo = systemctl enable algo

No inicies Xorg como root porque vas a romper el .Xauthority XD. Lo tuve que arreglar con: chown esotericwarfare .Xauthority.
