Android

Abrir links en tu Android TV desde Arch Linux por ADB

Paso 1: Instalar ADB en Arch Linux

sudo pacman -S android-tools

Paso 2: Activar ADB en el TV

  1. En el TV ir a Configuración → Preferencias del dispositivo → Acerca de
  2. Buscar “Número de compilación” y presionarlo 7 veces hasta que diga “Ya eres desarrollador”
  3. Volver a Configuración → entrar a Opciones de desarrollador
  4. Activar “Depuración por USB” (o cualquier opción de depuración disponible)

Paso 3: Conectar desde la PC

adb connect 192.168.0.230
adb connect 192.168.0.230:5555
adb connect 192.168.0.230:4321
adb connect 192.168.0.230:7777
adb connect 192.168.0.230:5555

El TV va a mostrar un cuadro pidiendo que aceptes la conexión — confirmarlo con el control remoto.

Paso 4: Abrir un link en el TV

adb shell am start -a android.intent.action.VIEW -d "https://ejemplo.com"

Uso diario

adb connect 192.168.0.230:5555 && adb shell am start -a android.intent.action.VIEW -d "https://tulink.com"

O un alias en .bashrc / .zshrc:

alias tv='function _tv(){ adb connect 192.168.0.230:5555 && adb shell am start -a android.intent.action.VIEW -d "$1"; }; _tv'

adb shell am start -a android.intent.action.VIEW -d "https://deportelibree.com/en-vivo-online/dazn-formula-1-es/embed.php"

Uso:

tv "https://youtube.com/watch?v=xyz"

alias tv='function _tv(){ adb connect 192.168.0.230:5555 && adb shell am kill-all && adb shell input keyevent KEYCODE_HOME && adb shell am start -a android.intent.action.VIEW -d "$1"; }; _tv'

adb shell am kill-all && adb shell input keyevent KEYCODE_HOME

adb shell ps | grep -v root

adb shell am kill-all

Con ADB en tu Smart TV podés hacer un montón de cosas:

📱 Control remoto desde terminal

# Subir/bajar volumen
adb shell input keyevent KEYCODE_VOLUME_UP
adb shell input keyevent KEYCODE_VOLUME_DOWN

# Pause/Play
adb shell input keyevent KEYCODE_MEDIA_PLAY_PAUSE

# Apagar/prender pantalla
adb shell input keyevent KEYCODE_POWER

📦 Instalar APKs (apps no disponibles en tu tienda)

adb install mi-app.apk

🗑️ Desinstalar bloatware

# Ver todas las apps instaladas
adb shell pm list packages

# Desinstalar una app del sistema (sin root)
adb shell pm uninstall --user 0 com.app.bloatware

📸 Captura de pantalla

adb shell screencap /sdcard/screen.png && adb pull /sdcard/screen.png

🎥 Grabar la pantalla

adb shell screenrecord /sdcard/video.mp4
# Ctrl+C para parar
adb pull /sdcard/video.mp4

📊 Ver consumo de recursos

# CPU y memoria en tiempo real
adb shell top

# Memoria disponible
adb shell cat /proc/meminfo

🌐 Ver el tráfico de red / IPs conectadas

adb shell netstat

🔤 Escribir texto directamente

adb shell input text "hola mundo"

🖱️ Simular clicks en coordenadas

adb shell input tap 500 300

💡 Alias útiles para agregar

alias tvshot='adb connect 192.168.0.230:5555 && adb shell screencap /sdcard/screen.png && adb pull /sdcard/screen.png'
alias tvvol+='adb connect 192.168.0.230:5555 && adb shell input keyevent KEYCODE_VOLUME_UP'
alias tvvol-='adb connect 192.168.0.230:5555 && adb shell input keyevent KEYCODE_VOLUME_DOWN'
alias tvpause='adb connect 192.168.0.230:5555 && adb shell input keyevent KEYCODE_MEDIA_PLAY_PAUSE'
alias tvpkg='adb connect 192.168.0.230:5555 && adb shell pm list packages'

alias dazn='adb connect 192.168.0.230:5555 && adb shell input tap 960 540 && sleep 2 && adb shell input tap 1420 1050'
