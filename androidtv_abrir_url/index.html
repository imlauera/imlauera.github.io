Android TV abrir URL desde PC

May 22, 2026
Android

Abrir links en tu Android TV desde Arch Linux por ADB

Paso 1: Instalar ADB en Arch Linux

sudo pacman -S android-tools

Paso 2: Activar ADB en el TV

  1. En el TV ir a Configuración → Preferencias del dispositivo → Acerca de
  2. Buscar “Número de compilación” y presionarlo 7 veces hasta que diga “Ya eres desarrollador”
  3. Volver a Configuración → entrar a Opciones de desarrollador
  4. Activar “Depuración por USB” (o cualquier opción de depuración disponible)

Paso 3: Conectar desde la PC

adb connect 192.168.0.230
adb connect 192.168.0.230:5555
adb connect 192.168.0.230:4321
adb connect 192.168.0.230:7777
adb connect 192.168.0.230:5555

El TV va a mostrar un cuadro pidiendo que aceptes la conexión — confirmarlo con el control remoto.

Paso 4: Abrir un link en el TV

adb shell am start -a android.intent.action.VIEW -d "https://ejemplo.com"

Uso diario

adb connect 192.168.0.230:5555 && adb shell am start -a android.intent.action.VIEW -d "https://tulink.com"

O un alias en .bashrc / .zshrc:

alias tv='function _tv(){ adb connect 192.168.0.230:5555 && adb shell am start -a android.intent.action.VIEW -d "$1"; }; _tv'

adb shell am start -a android.intent.action.VIEW -d "https://deportelibree.com/en-vivo-online/dazn-formula-1-es/embed.php"

Uso:

tv "https://youtube.com/watch?v=xyz"

alias tv='function _tv(){ adb connect 192.168.0.230:5555 && adb shell am kill-all && adb shell input keyevent KEYCODE_HOME && adb shell am start -a android.intent.action.VIEW -d "$1"; }; _tv'

adb shell am kill-all && adb shell input keyevent KEYCODE_HOME

adb shell ps | grep -v root

adb shell am kill-all
