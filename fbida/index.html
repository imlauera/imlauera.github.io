Fbida: ver imágenes, pdf en la TTY.

November 18, 2024
Fbida

Fbida instalará fbi, ida, fbpdf. O también podés usar fim que es una versión mejorada fbi te hace un autozoom con yay -S fim

Actualización diciembre 2025: No necesitás usar fim podes usar fbi --autozoom --edit *.jpg que es mejor.

Ejemplo de uso.

fbi --autozoom --edit *.jpg
fbpdf *.pdf

Se puede usar las keybindings de vim para ver diferentes imágenes.
