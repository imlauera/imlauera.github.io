Iniciar TMUX en el login en la TTY

November 19, 2024
Linux

Con fish

En el .config/fish/fish.conf agregá:

if status --is-login
  if test -z "$DISPLAY" -a $XDG_VTNR = 1
    tmux attach-session -t mysession 2>/dev/null || tmux new-session -s mysession
  end
end

Con bash

if [ -z "$TMUX" ] && [ -n "$SSH_TTY" ] && [[ $- =~ i ]]; then
  tmux attach-session -t mysession 2>/dev/null || tmux new-session -s mysession
fi
