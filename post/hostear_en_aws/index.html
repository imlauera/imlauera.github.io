  1. Logueate en la consola AWS.
  2. Crea una máquina ec2.
  3. Configurá security group(ssh -22 admin only)(80 - for normal traffic)
  4. Conectate a la máquina ec2.
  5. Instalá el servidor web httpd(apache) o podés instalar ngnix para correr la aplicación web.
sudo su
yum update -y
yum install httpd -y
cd /var/www/html
service httpd start
  1. Creá tu página en ese directorio.
  2. Acedé a tu página desde el navegador usando la IP pública de la máquina ec2 o del DNS.
