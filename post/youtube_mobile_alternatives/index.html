Como mirar YouTube en Android sin publicidad

April 20, 2023
Quitar Publicidad Android Youtube Ads Remove Premium

Alternativas a YouTube sin publicidad.

Desde la F-droid pueden bajarse:

  1. LibreTube (si el video carga lento intenta cambiando de instancias en el menu de ajustes)
  2. NewPipe Observación: Estos dos clientes no tienen el algoritmo de recomendacion de videos, pero funcionan bien.

La siguiente aplicacion no se puede bajar desde la f-droid, es la mas completa y tiene el algoritmo de recomendacion de videos YouTube Revanced

La forma oficial de instalar ReVanced

Bajando el apk de https://github.com/revanced/revanced-manager/releases/ necesitan MicroG como pre-requisito (esta en el github de https://github.com/revanced/revanced-manager el link de descarga) y parcheando la aplicacion de YouTube deben habilitar el parche MicroG que esta a lo ultimo (para que el parche funcione tiene que tener una version que sea admitida por el manager), tenes que ver para que version de YouTube funcionan LA GRAN MAYORIA de parches y bajar esa version de YouTube desde ApkMirror (que es confiable) (la version recomendada te la dice cuando abris Revanced Manager) si tienen una version de YouTube muy nueva tienen que desinstalar la actualizacion. Luego parchan desde Revanced Manager ESA version de YouTube que bajaron desde ApkMirror y es compatible con RevancedManager (la actual es 18.15.40). Ojo: despues de instalar el apk de YouTube desde APKMirror tenes que desabilitar la aplicacion de YouTube y desactivar Google Play Protect (despues de la instalacion lo pueden volver a activar), luego abrir RevancedManager seleccionar YouTube e seleccionar los parches que te gusten e instalarla.

FORMA NO OFICIAL (POSIBLE MALWARE???)

No se recomienda hacerlo de esta forma porque no es la oficial, es un APK generado por un chinardo que le pudo haber agregado spyware/malware. Usando https://revanced.io

Si tu telefono no esta rooteado

Si tienen un Android no rooteado tienen que descargar MicroG que es una micro implementacion de los google services: https://github.com/inotia00/VancedMicroG/releases y use el repositorio de j-hc para descargar el apk de revanced (actualmente se llama youtube-revanced-v18.05.40-all.apk). Aca estan las diferentes publicaciones https://github.com/j-hc/revanced-magisk-module/releases Observacion: El unico problema que hay al usar MicroG es que no funciona el chrome cast.

Si tu telefono esta rooteado usando MAGISK SU

Si esta rooteado con MAGISK SU (si no saben lo que es pueden buscarlo en YouTube) entonces pueden descargar el Magistik module que se llama mindetach.zip https://github.com/j-hc/revanced-magisk-module/releases/ y no necesitan instalar MicroG y aca si funciona el envio al chromecast. Y el zip de YouTube Revanced que lo tienen que instalar con Magisk que se llama youtube-revanced-magisk-all.zip https://github.com/j-hc/revanced-magisk-module/releases/

Aca lo tienen mas aclarado como instalar revanced usando MicroG (para android no rooteados): https://web.archive.org/web/20230203104412/https://revanced.io/#How_to_Install_ReVanced_YouTube_APK_on_Android Aca lo tienen mas aclarado como instalar revanced usando el magisk module: https://web.archive.org/web/20230203104412/https://revanced.io/#How_to_Install_YouTube_ReVanced_Magisk_Module_on_Rooted_Android
