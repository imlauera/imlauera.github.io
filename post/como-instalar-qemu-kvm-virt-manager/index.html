Como Instalar Qemu Kvm Virt Manager

Andres Imlauer
19.11.2022
Índice
sudo apt install -y virt-manager qemu qemu-kvm

Luego de reiniciar ejecutar

virt-manager

Y listo, mas rapido que VMWare y VirtualBox. Pueden probar instalar una distro de Linux pequeña como Alpine.
