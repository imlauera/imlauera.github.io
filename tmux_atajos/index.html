Tmux

Atajos de tmux

Nota: El prefijo por defecto es Ctrl+b (representado como prefix).

Sesiones

Tecla Acción
prefix d Detach de sesión
prefix D Detach otros clientes
prefix s Listar sessões (interactivo)
prefix $ Renombrar sesión
prefix ( Sesión anterior
prefix ) Siguiente sesión

Ventanas

Tecla Acción
prefix c Nueva ventana
prefix w Listar ventanas
prefix n Siguiente ventana
prefix p Ventana anterior
prefix l Última ventana
prefix , Renombrar ventana
prefix . Mover ventana
prefix & Cerrar ventana
prefix 0-9 Ir a ventana #

Panes

Tecla Acción
prefix " Split vertical
prefix % Split horizontal
prefix o Cycle panes
prefix ; Último pane
prefix ←↑→↓ Ir a pane
prefix x Cerrar pane
prefix z Zoom pane
prefix Space Cycle layouts
prefix q Mostrar números de pane
prefix ! Break pane a ventana

Resize de panes

Tecla Acción
prefix Ctrl+← Resize left
prefix Ctrl+→ Resize right
prefix Ctrl+↑ Resize up
prefix Ctrl+↓ Resize down

Copy Mode (vi mode)

Entrar: prefix + [

Navegación

Tecla Acción
h/j/k/l Moverse izq/abajo/arriba/derecha
w/b Palabra adelante/atrás
0/$ Inicio/fin de línea
g Ir al inicio del buffer
G Ir al final del buffer
Ctrl+u / Ctrl+d Página arriba/abajo
/ Buscar adelante
? Buscar atrás
n Siguiente match
N Match anterior

Selección y copia

Tecla Acción
Space Iniciar selección
v Modo visual (selección por caracteres)
Enter Copiar selección y salir
q Salir de copy mode

Pegar

Tecla Acción
prefix ] Pegar buffer
prefix = Listar buffers

Varios

Tecla Acción
prefix ? Ver todos los atajos
prefix : Línea de comandos
prefix t Reloj
prefix r Refrescar cliente
prefix :source-file ~/.tmux.conf Recargar config

Buffers

Comando Acción
tmux list-buffers Listar buffers
tmux save-buffer archivo.txt Guardar buffer a archivo
tmux show-buffer -b # Mostrar buffer #

Notas

  • Para cambiar el prefijo a Ctrl+a:

    unbind C-b
set -g prefix C-a

  • Para usar vi mode en copy mode, agregar a ~/.tmux.conf:

    setw -g mode-keys vi

  • Recargar config: prefix r (si está configurado)
