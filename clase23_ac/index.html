  1. En un ambiente de 5 metros x 8 metros donde concurren normalmente 20 personas. ¿Cuántos BTU se necesita para refrigerarlo?

Si es 2,6 metros es lado x lado x 200. Entonces tenemos: 5x8x200+20x70=9400 frigorías x 4 = 37600 BTU.

Es importantísimo poner la unidad al final (BTU).

  1. ¿Cómo se recupera el refrigerante de un equipo controlado con manifull? Esta la podemos hacer: a,b,c,d.

Primero encendemos el equipo, si no encendemos el equipo no podemos colocar el manómetro porque hay alta presión. Entonces:

a. Se enciende el equipo.
b. Se conecta el manómetro y ahí recién se va a comprimir.
c. Cierran robinete de alta.
d. Se cierra el de baja.
e. Se apaga el equipo.

  1. Cuándo se instala un equipo de aire acondicionado que operación debe realizar para habilitar el refrigerante.

Una vez instalado el equipo, conecto el manifull al robinete de servicio y a la bomba saco humedad e impureza del sistema para volver a habilitar el refrigerante. Ahí abrimos el robinete de alta y luego el de baja.
