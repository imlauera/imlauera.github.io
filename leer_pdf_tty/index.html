Leer pdf desde la tty

November 17, 2024
Linux

Ya expliqué como mirar imágenes desde la TTY usando el navegador w3m y descargá el PDF. Expliqué como mirar videos de YouTube usando mpv y ytfzf. También expliqué como buscar torrents desde la consola. Se puede usar MPV para mirar videos y para mirar imágenes.

Esta es mi actual configuración de /etc/mpv/mpv.conf:

[I] esotericwarfare@arch ~> cat /etc/mpv/mpv.conf
--ytdl-raw-options="format=bestvideo[height<=?360]+bestaudio/best[height<=?360]/bestvideo[height<=?480]+bestaudio/best[height<=?480],write-auto-sub=,sub-lang=[en,ru],write-sub="
#--vo=drm
[I] esotericwarfare@arch ~>

Y este es mi /etc/yt-dlp.conf:

[I] esotericwarfare@arch ~> cat /etc/yt-dlp.conf
--format=18
--write-auto-sub
--write-sub
--sub-lang=en,ru,es
--cookies-from-browser=chromium
[I] esotericwarfare@arch ~>

Leer pdf’s desde la terminal: https://bbs.archlinux.org/viewtopic.php?id=72913

sudo pacman -S fbida # Instala fbi, fbgs, ida y exiftran
fbpdf /tmp/mipdf.pdf
