📡 Apps de código abierto para comunicarte sin internet

1. Briar (ya la conocés)

Permite mensajería entre usuarios cercanos mediante Bluetooth o WiFi Direct. Tiene cifrado extremo a extremo, es totalmente descentralizada y no rastrea al usuario. Cuando hay internet, usa la red Tor para mayor anonimato. Solo disponible para Android.

2. Bitchat (muy nuevo, de Jack Dorsey)

Se basa en Bluetooth Low Energy (BLE) mesh networking. Los dispositivos se conectan directamente entre sí mediante señales cifradas sin pasar por servidores. Si dos personas están fuera del alcance convencional (~100 metros), otros teléfonos intermedios actúan como “puentes”, extendiendo el alcance efectivo a más de 300 metros. El código es abierto y auditables en GitHub. Su debilidad: Bluetooth LE prioriza eficiencia energética, no velocidad; texto sí, pero voz, video o archivos grandes quedan fuera.

3. Meshtastic ⭐ (el más poderoso para tu zona)

Esta es la joya para alguien en Corrientes. Meshtastic es un proyecto de código abierto que combina apps móviles con dispositivos de radiofrecuencia LoRa para crear una red mesh descentralizada. Entre el nodo y el teléfono se usa Bluetooth Low Energy (BLE); entre nodos, se usa LoRa (radio) directamente.

En condiciones ideales, los nodos pueden comunicarse entre 2 y 5 km, y entusiastas han logrado récords de más de 300 km usando antenas especializadas en altitud. Los dispositivos están pensados para operar con batería por largos períodos, pudiendo mantener nodos activos durante días o semanas.

En festivales o eventos masivos donde la red móvil se satura, o en contextos de censura donde se apagan las redes intencionalmente, Meshtastic permite mantener canales alternativos. No es sustituto del internet ni sirve para videollamadas o fotos pesadas, pero para texto y GPS es ideal.

Hardware necesario: dispositivos como T-Beam, T-Echo o Heltec, que cuestan entre $20-50 USD en AliExpress.

4. Bridgefy

Aunque permite mandar mensajes por Bluetooth, crea una red de malla que permite que los mensajes vayan utilizando a los usuarios como nodos para llegar más lejos. Está disponible en Google Play y App Store, pero requiere una cuenta de Gmail para registrarse. Es menos libre que las anteriores (requiere cuenta).

🧠 El sueño: LLM local sin internet

Tu idea es totalmente viable hoy en 2026. Correr modelos de lenguaje en tu propia máquina es hoy completamente alcanzable para cualquiera con una computadora medianamente decente. Con herramientas como Ollama o LM Studio, en 10 minutos tenés un LLM funcionando localmente, completamente privado, sin conexión a internet.

Herramientas principales

Ollama — la más simple Con un simple comando podés descargar y usar modelos como Llama 3, Mistral, Gemma, Phi-3 y muchos más. Funciona en macOS, Linux y Windows, y tiene una API local para integrar en proyectos con Node.js, Python u otros lenguajes.

LM Studio — sin tocar la terminal Ideal si preferís cliquear en vez de escribir comandos. Muestra el VRAM usado en tiempo real, permite ajustar parámetros visualmente y tiene un chat integrado para probar modelos. La curva de aprendizaje es prácticamente cero.

llama.cpp — para usuarios avanzados Un proyecto open source para ejecutar localmente modelos basados en LLaMA. Puede aprovechar tanto CPU como GPU. Es lo que Ollama usa por debajo, pero con todos los controles expuestos.

¿Qué hardware necesitás?

Modelos como Llama 3.2, Mistral, Qwen y Gemma 2 funcionan en CPUs, pero una GPU NVIDIA acelera 10x la generación de texto. Necesitás mínimo 8-16 GB de RAM para modelos pequeños (7B parámetros), 32+ GB para medianos (34B) y 48+ GB para grandes (70B+).

🏆 Mi recomendación para vos, en Corrientes

Teniendo en cuenta que estás en zona del litoral argentino con zonas rurales extensas:

  1. Meshtastic es tu mejor apuesta para comunicación real a distancia. Con 2-3 nodos solares estratégicamente ubicados podés cubrir una zona enorme sin ningún costo mensual.
  2. Para LLM local, empezá con Ollama + Mistral 7B si tenés 8-16 GB de RAM. Si tenés menos, probá Phi-3 Mini (3.8B), que corre hasta en computadoras básicas.
  3. Briar para mensajería cotidiana entre personas cercanas.
