Modificar Makefile usando yay

May 02, 2025
Linux

  1. Download the build files without installing:

    yay -G libgbinder

  2. Navigate into the package directory:

    cd libgbinder

  3. Modify the Makefile: Look for the Makefile in the source directory. You’ll need to wait until the sources are extracted:

    makepkg -o

    This will download and extract the source files without building, so you can modify them.

  4. Edit the Makefile: Now go into the extracted source directory (usually inside src/):

    cd src/libgbinder-<version>

    vim Makefile

    Make your changes and save.

  5. Build the package: Go back to the main package directory:

    cd ../../
makepkg -si
